لوحة الإدارة

دليل استخدام لوحة التحكم الإدارية

الوصول

رابط لوحة الإدارة:

https://your-domain.com/admin/login

بيانات الدخول الافتراضية:

  • البريد: admin@mytrip.com
  • كلمة المرور: password

تحذير: يجب تغيير كلمة المرور فوراً بعد أول تسجيل دخول!

لوحة المعلومات

الصفحة الرئيسية تعرض:

  • إجمالي عدد المستخدمين
  • عدد الأدوار النشطة
  • الحجوزات الجديدة
  • إحصائيات سريعة

إدارة المستخدمين

عرض المستخدمين

المسار: /admin/users

  • قائمة جميع المستخدمين
  • البحث والتصفية
  • عرض التفاصيل

إضافة مستخدم

المسار: /admin/users/create

  • إدخال البيانات الأساسية
  • تحديد النوع (Admin/Customer)
  • تعيين الأدوار

تعديل مستخدم

المسار: /admin/users/{id}/edit

  • تحديث البيانات
  • تغيير الحالة
  • إدارة الأدوار

الأدوار والصلاحيات

إدارة الأدوار

المسار: /admin/roles

  • إنشاء أدوار جديدة
  • تعديل الأدوار الموجودة
  • تعيين الصلاحيات للأدوار
  • حذف الأدوار غير المستخدمة

إدارة الصلاحيات

المسار: /admin/permissions

  • إنشاء صلاحيات جديدة
  • تعديل الصلاحيات
  • ربط الصلاحيات بالأدوار

الأدوار الافتراضية:

الدور الوصف الصلاحيات
Admin مدير النظام جميع الصلاحيات
Customer عميل الوصول للتطبيق فقط

الإعدادات

المسار: /admin/settings

الإعدادات المتاحة:

الإعداد الوصف
اسم الموقع (عربي/إنجليزي) يظهر في التطبيق والبريد
الوصف (عربي/إنجليزي) وصف المنصة
اللون الأساسي لون العلامة التجارية
وضع الصيانة تفعيل/تعطيل الموقع
شعار المنصة رفع الشعار
أيقونة المتصفح Favicon
بريد الدعم للتواصل
روابط التواصل الاجتماعي Facebook, Twitter, Instagram
