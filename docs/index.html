عن المشروع

My Trip هي منصة متكاملة لإدارة الحجوزات السياحية تشمل حجوزات الفنادق والرحلات الجوية. تم بناء المشروع باستخدام Laravel 12 مع واجهة إدارة حديثة وAPI RESTful للتطبيقات المحمولة.

المميزات الرئيسية

نظام مصادقة متقدم

  • تسجيل وتسجيل دخول بنظام OTP
  • Laravel Sanctum للـ API
  • إدارة الأدوار والصلاحيات (Spatie)
  • تحقق بخطوتين عبر البريد

بوابة الدفع

  • تكامل مع HyperPay
  • دعم Mada و Visa/Mastercard
  • Apple Pay جاهز
  • تحقق آمن من المدفوعات

الإشعارات الفورية

  • Firebase Cloud Messaging
  • إشعارات Push للجوال
  • نظام بريد إلكتروني متقدم
  • قوالب بريد احترافية

دعم متعدد اللغات

  • العربية والإنجليزية
  • ترجمة كاملة للـ API
  • ترجمة رسائل التحقق
  • واجهة إدارة ثنائية اللغة

توثيق Swagger

  • توثيق تفاعلي للـ API
  • اختبار مباشر للنقاط
  • أمثلة واقعية
  • دعم المصادقة

الأمان

  • تشفير كلمات المرور
  • حماية CSRF
  • Rate Limiting
  • Validation شامل
