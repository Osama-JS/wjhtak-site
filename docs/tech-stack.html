البنية المعمارية

MVC Pattern

يتبع المشروع نمط MVC (Model-View-Controller):

  • Models: تمثيل البيانات والعلاقات (Eloquent ORM)
  • Views: واجهات المستخدم (Blade Templates)
  • Controllers: منطق التطبيق وإدارة الطلبات

Service Layer

طبقة خدمات منفصلة لمنطق الأعمال:

  • MailService - إدارة البريد الإلكتروني
  • FirebaseService - إدارة الإشعارات
  • HyperPayService - إدارة المدفوعات

Repository Pattern

استخدام Eloquent كطبقة تجريد لقاعدة البيانات
